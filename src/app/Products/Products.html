Cost Estimator is a software designed to help importers estimate the cost of importing any specific type of cargo.
The advantage is that it allows importers plan their finances regarding cargo import or perhaps check alternative sources for the cargo in order to minimize cost

  • For any cargo type(by HS code).
  • Calculate freight charges
  • Calculate port charges
  • Calculate taxes
